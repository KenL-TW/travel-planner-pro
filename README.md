# Travel Planner Pro (Streamlit)

## Features
- Trip / Day / Event CRUD
- Task assignment (member, status, priority, due date)
- Task Board with filters (keyword/category/status/priority/assignee)
- Team management (create member, add/remove from trip)
- Checklists (multi-list + items CRUD)
- SQLite persistence + JSON export/import

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
